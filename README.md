# Full-Stack Application - TP 8 Web

Projet d'application Full-Stack implémentant une architecture découplée (Frontend React / Backend Node.js Express) avec un système d'authentification par jeton sécurisé (JWT) et simulation d'accès aux données.

## 📁 Structure du Projet

- `frontend/` : Application cliente (interface utilisateur React).
- `backend/` : API REST de production gérant l'authentification et les routes sécurisées.
